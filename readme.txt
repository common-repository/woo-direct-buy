This plugin allow direct buy to woocommerce. NO Shopping Cart. Just one product per purchase.

== Description ==

Checkout Without Shopping Cart.
This plugin allow direct buy to woocommerce, perfect to marketplaces like "Mercadolibre"
This show a  "Buy now" instead a "Add to cart"
This redirect user to checkout page after click "Buy now button"
This dont allow multiples products to cart, just one product per transaction.
When user click on buy now, just the current product keep in cart.


== Installation ==


1. Upload `woo_direct_buy` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
